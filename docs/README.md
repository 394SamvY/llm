## 🔑 核心概念速查

### 两种训诂类型

| 类型 | 特点 | 判断依据 |
|------|------|----------|
| **假借说明** | 音近**义远** | 字的替代，借字与正字语义无关 |
| **语义解释** | 音近**义近** | 词义解释，揭示语源 |

### 五步推理

| 步骤 | 判断内容 | 工具 |
|------|----------|------|
| Step 1 | 义近/义远 | 字典查询 |
| Step 2 | 音近/音远 | 音韵查询 |
| Step 3 | 有/无异文 | 文献检索 |
| Step 4 | 训式类型 | 正则匹配 |
| Step 5 | 语境适配 | LLM分析 |

### 典型示例

```
"崇，终也" → 假借说明
  - 崇(高大) ≠ 终(终结) → 义远
  - 崇/终 上古音东部 → 音近
  - 《小雅》有"终朝"异文 → 有佐证
  - 结论：假借说明

"海，晦也" → 语义解释（需验证）
  - 海因晦暗(黑色)得名 → 义近
  - 海/晦 上古音相近 → 音近
  - 结论：语义解释/以声通义
```
